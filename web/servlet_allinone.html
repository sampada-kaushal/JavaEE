Servlet all in One


Here we will be creating a project in which we will have index file containing 2 fields username and password.
The user have to fill the name and password and have to press the enter.
The control will be than transferred to web.xml file .
Inside the web.xml file url pattern would be matched and servlet login.java will be opened.Inside the login.java file username and password will be matched with the already enterd data if the data is matched then welcome.java file will be opened otherwise the message will be printed.
The message that is to be printed is stored in web.xml under the tag "init param".Inside login we have also used session.


Index.jsp
As stated above the index file contain form action "login" and 2 fields username and password. When user press submit control goes to web.xml.


Web.xml
This file contains servlet mapping tags and contex and config parameters.


Login.java
This file contains HttpSession object, RequestDispatcher, ServletConfig object and ServletContext object.


Welcome.java
This page uses the session object.Ths is to remember that only first session is true and other session are always false.
