Table Per Hierarchy


Steps to use Table Per Hierarchy in Hibernate Inheritance

1. Suppose we are making an application of Employee data; we need to create a table named EMPDATA with the following parameters:
* Empdata Table in DataBase

 

2. Now a POJO class is made with setters and getters, containing parameters as int id and String name. Name that file as Employee.java
* Employee.java

 

3. It is time to create another POJO class named as Regular_Employee.java with parameters int salary and int bonus. Create its setters and getters.
* Regular_Employee.java

 

4. Make another POJO class named as Contract_Employee.java with parameters pay_per_hour and String duration. Create its setters and getters.
* Contract_Employee.java

 

5. Create the servlet for configuring these POJO class' objects. Enter the values and save it to database.
* Servlet.java

 

6. In hibernate.cfg.xml, add mapping resource by giving the name of mapping xml.
* hibernate.cfg.xml

 

7. Enter the objects' name and their properties in mapping xml.
* hibernate.hbm.xml

 

 

* Output

 

  Download example and run it on Netbeans IDE
