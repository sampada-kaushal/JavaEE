Hidden Form Field
Hidden fields are similar to text fields, with one very important difference!
The difference is that the hidden field does not show on the page. Therefore the visitor can't type anything into a hidden field,
which leads to the purpose of the field:
To submit information that is not entered by the visitor.

Use
1. You can use hidden form fields to maintain the session information of a user while the user interacts with the web application.
2. A hidden form field is embedded in an HTML page and is not visible when viewed in a browser.
