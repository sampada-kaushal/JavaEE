Dependent Objects

Following example of Spring depicting Dependent Objects.

Steps to create Spring Application Using Eclipse IDE
1. Create Pojo class or class having constructor.(These files can be multiple.)
2. Create XML file with Beans.
3. Create Main class

1. Create two classes with constructors:
i. First make a class named CompanyDetail.
ii.Take String variables add(Address) and companyname.
iii.Create two methods in it;
-constructor, CompanyDetail(), with parameters as String add and companyname and
-method toString() to return the companyname and add.

CompanyDetail.java


iv. Make another class JavaDeveloper.
v. Take variables-int id,String projectname,String name and instance of class CompanyDetail.
vi. Create an empty constructor, a parameterised constructor and a method display() to see the output.

JavaDeveloper.java


2. Create XML:
Make XML file defining object's name and corresponding constructor value arguments.

dependentObjectBean.xml


3. Create main class:
i.Create a java class named Profile containing main() method.
ii.In the main() method, create an instance of Resource and the path of above created XML.
iii.Create an instance of BeanFactory to use Resource instance.

Profile.java
