Collection Injection

We can inject values in constructor using collection framework.
like list,set,map.
Here we are using List .
1. Create class JavaDeveloper with two methods.
a) Constructor JavaDeveloper with 5 parameters.
b) Display method to display the values .

                           


2. Create xml file with name as SpringBean.xml.
This file contains the name of the object ,we want container to make for us.

Since we are using collection so we will use list tag .
Inside list tag we will assign the data to List object inside the value tag.


                              


3. Create the class with main method to access the object and methods .
